# Miami-Dade Wholesale Property Research Instructions

## Property Appraiser Search

Use the Miami-Dade Property Appraiser website to gather ownership, equity, and absentee owner information.

[Miami-Dade Property Search](https://www.miamidadepa.gov/pa/real-estate/property-search.page)

### Instructions

1. Open the Property Search website
2. Search using:

   * Folio number
   * OR property address

Example folio:

```text id="8m2qvp"
01-0101-050-1080
```

Example address:

```text id="7x4nkr"
243 NW 10 ST
```

3. Review the property details.

# Important Data To Extract

## Ownership Information

Look for:

* Owner name
* Mailing address
* Property address
* Years owned
* Trust/LLC ownership

## Absentee Owner Detection

Compare:

```text id="4v8qtm"
Property Address
VS
Mailing Address
```

If they are different:

```text id="1p7mzl"
Absentee Owner = TRUE
```

Example:

```text id="2n5qvk"
Property Address:
243 NW 10 ST

Mailing Address:
8300 BISCAYNE BLVD
```

This indicates a likely absentee owner.

---

# Strong Absentee Signals

```text id="5k3nqp"
Different mailing address
Different city
Different state
Trust ownership
LLC ownership
Long-term ownership
```

---

# Equity Indicators

Look for:

* Long ownership history
* Large difference between assessed value and market value
* Older purchase dates
* Paid-off properties
* Non-homestead exemption

Strong example:

```text id="9x2mkr"
Owned since 2000
```

This usually indicates strong equity.

---

# Official Records Search

Use the Miami-Dade Official Records website to identify:

* probate
* foreclosure
* liens
* judgments
* lawsuits
* divorce
* distress indicators

[Miami-Dade Official Records Search](https://onlineservices.miamidadeclerk.gov/officialrecords)

## Instructions

1. Open the website
2. Click:

```text id="6q8vnp"
Standard Search
```

3. Click:

```text id="2m4kzr"
Property/Condo
```

4. Enter ONLY the property address.
   Do NOT include:

* city
* state
* zip code

Example:

```text id="5v1qtm"
243 NW 10 ST
```

5. Click:

```text id="8r2qvx"
Search
```
6. 
search for only these high-value wholesale lead indicators in the search results:

# High-Value Wholesale Lead Document Types (8/10+)

```text id="1z9mvl"
LIS - Lis Pendens
NTL - Notice of Tax Lien
LIE - Lien
FTL - Federal Tax Lien
PAD - Probate & Administration
BAN - Bankruptcy
JUD - Judgment
LNJUD - Any Lien Judgment
CVP - Civil Court Paper
QCD - Quit Claim Deed
DOM - Dissolution of Marriage
PRO - Probate Order of Distribution
AJ - Affidavit with Judgment Attached
```
7. Review all documents attached to the property.

# Code Enforcement Search

Use the Miami-Dade Code Enforcement systems to identify:

* unsafe structures
* abandoned property
* illegal work
* open violations
* unpaid fines
* distressed property conditions

[Miami-Dade Regulation Cases Search](https://www.miamidade.gov/Apps/RER/RegulationSupportWebViewer/?utm_source=chatgpt.com)

## Instructions

1. Open the website
2. Search using:

* property address
* folio number
* owner name if necessary

Example:

```text id="7k3qmp"
243 NW 10 ST
```

or:

```text id="4n8vzr"
01-0101-050-1080
```

# Look For

```text id="6p2mqt"
Unsafe structures
Expired permits
Work without permits
Open code violations
Abandoned property
Large fines
Repeat violations
```

Properties with:

* multiple violations
* unresolved violations
* city fines
* unsafe structure cases

are much stronger wholesale lead indicators.

---

# Strong Combo Signals

These combinations are some of the strongest wholesale indicators:

```text id="9m4qvk"
PAD + absentee owner
LIS + high equity
FTL + distressed property photo
DOM + long ownership
QCD + vacant property
BAN + code violations
NTL + distressed property
JUD + absentee owner
CVP + neglected property
Code violations + absentee owner
Unsafe structure + long ownership
Liens + distressed property
```

These combinations often indicate:

* motivated sellers
* financial distress
* inherited properties
* legal pressure
* distressed ownership
* potential wholesale opportunities.
